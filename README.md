# Getting Started

Follow these steps to get the project up and running on your local machine.

---

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your system.

### Setup

Edit public/index.html, replace TOKEN,COMPANY-ID, LOCATION-ID

### Installation

First, install the necessary dependencies using the Node Package Manager:

```bash
npm install

```

### Launch the Application

Once the installation is complete, you can start the development server:

```bash
npm start

```

---

> **Note:** The application should now be running. Check your terminal output for the local URL (usually `http://localhost:3000`).
